# 04. Sum of n integers using loop



```
Input: n
Result = 1 + 2 + 3 + ... + n
Output: result
```

```cpp
#include <bits/stdc++.h>
using namespace std;

int main()
{
  int n;
  cout << "n: ";
  cin >> n;

  int res = 0;
  for (int i = 1; i <= n; i++)
    res += i;

  cout << res;
  return 0;
}
```
