# 02. Reverse an Array

[Question Link](https://practice.geeksforgeeks.org/problems/reverse-an-array/0/)

### Solution

Print the array from end to start.

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  int T;
  cin >> T;

  while(T--) {
    int n;
    cin >> n;

    vector<int> arr(n);
    for(int i = 0; i < n; i++)
        cin >> arr[i];

    for(int i = n - 1; i >= 0; i--)
      cout << arr[i] << ' ';

    cout << endl;
  }
  return 0;
}
```
