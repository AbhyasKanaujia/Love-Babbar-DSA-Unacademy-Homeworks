# Functions

### String

| Function                      | Remark                                                | Time Complexity |
| ----------------------------- | ----------------------------------------------------- | --------------- |
| `int stoi(string str)`        | Convert Given String to Integer                       | O(n)            |
| `stol()`, `stoll()`, `stod()` | Convert Given string to `long`, `long long`, `double` | O(n)            |
|                               |                                                       |                 |
|                               |                                                       |                 |
